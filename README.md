## jgmenu  
  
simple X11 menu  
  
requires:    
```
apt install jgmenu
```  
```
yum install jgmenu
```  
```
pacman -S jgmenu
```  
  
Automatic install/update:
```
bash -c "$(curl -LSs https://github.com/dfmgr/jgmenu/raw/master/install.sh)"
```
Manual install:
```
mv -fv "$HOME/.config/jgmenu" "$HOME/.config/jgmenu.bak"
git clone https://github.com/dfmgr/jgmenu "$HOME/.config/jgmenu"
```
